# Промпт: Заполнение контекста проекта

Используйте этот промпт с AI-агентом (Cursor, Copilot, Claude) для заполнения `context.md`.

---

## Промпт

```
Помоги мне заполнить контекст моего проекта для маркетингового анализа.

Я буду отвечать на твои вопросы, а ты структурируй информацию в формате markdown.

Задавай вопросы по одному, копай глубже если ответ слишком абстрактный.

Начни с вопроса: "Чем занимается твоя компания/проект?"

После всех вопросов сформируй готовый текст для файла context.md по этой структуре:

## О компании
[2-3 предложения]

## Продукт/услуга  
[Что продаёте]

## Целевая аудитория
[Кто покупает, какие сегменты]

## Текущая ситуация
[Выручка, клиенты, каналы]

## Конкуренты
[2-3 конкурента и отличия]

## Цели
[На 3 мес и год]

## Ограничения
[Бюджет, ресурсы, время]
```

---

## Примерный диалог

**AI:** Чем занимается твоя компания/проект?

**Вы:** Продаём онлайн-курсы для предпринимателей

**AI:** Какие именно курсы? Чему учите?

**Вы:** Учим делать growth-эксперименты

**AI:** Кто ваш типичный клиент? Опишите его.

**Вы:** Владельцы малого бизнеса, у которых уже есть продукт...

*И так далее*

---

## Советы

- Отвечайте конкретно, с цифрами
- Не бойтесь говорить о проблемах
- Если не знаете — так и скажите
- Лучше меньше, но точнее
